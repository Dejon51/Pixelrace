# PixelRace

## Overview
PixelRace is a retro-style racing game built in Python that runs entirely in the Windows terminal. It renders BMP images as ANSI-colored graphics, creating a pixel-art driving experience with real-time movement, animation, and sound.

---

## Features
- Terminal-based graphics using BMP rendering
- Layer system with transparency support
- Smooth car movement with acceleration + friction
- 8-direction car sprites
- Animated menu + intro cutscene
- Camera-follow gameplay system
- WAV sound playback

---

## Requirements
- Python 3.10+
- Windows OS (uses `msvcrt` + `winsound`)
